# Arcane Nexus
Help for PHP developers working with WordPress.

**RU**

**Arcane Nexus** — это плагин, расширяющий возможности PHP-разработки при работе с WordPress.

## Установка:
Установка производится типичными для WordPress методами загрузки сторонних плагинов: можно либо создать папку для плагина в _/wp-content/plugins/_ и скопировать в неё содержимое плагина, либо воспользоваться функцией "Загрузить плагин" в админпанели WordPress и указать заранее скачанный на компьютер файл с архивом плагина.
